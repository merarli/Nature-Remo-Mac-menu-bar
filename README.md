# Nature Remo Mac Menu Bar

## 概要
Nature Remo APIを使用して温度、湿度、照明の明るさ、絶対湿度をMacのメニューバーに表示することができます。

## セットアップ
.envファイルを作成後
```
$ touch .env 
```

.envファイルにNature RemoのAPI KEYの入力してください。
```
NATURE-REMO-API-KEY=hogehogehoge
```

必要なライブラリをインストールします。
```
$ npm install
```

## 実行

```
$ npm run start
```

## ビルド
```
$ npm run build
```
